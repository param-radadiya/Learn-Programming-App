fun main(args: Array<String>) {
    val year = 1900
    var leap = false
    if (year % 4 == 0) {
        if (year % 100 == 0) {
            // year is divisible by 400, hence the year is a leap year
            leap = year % 400 == 0
        } else
            leap = true
    } else
        leap = false
    println(if (leap) "$year is a leap year." else "$year is not a leap year.")
}


>> output will be:

1900 is not a leap year.
========================================
In the above program, given year 1900 is 
stored in the variable year.

Since 1900 is divisble by 4 and is 
also a century year (ending with 00), it 
has be divisble by 400 for a leap year. 
Since it's not divisible by 400, 1900 is 
not a leap year.

But, if we change the year to 2000, 
it is divisible by 4, is a century year 
and is also divisible by 400. So, 2000 
is a leap year.

Likewise, If we change the year to 
2012, it is divisible by 4 and is not a 
century year, so 2012 a leap year. We 
don't need to check if 2012 is divisible 
by 400 or not.
